# Gottam Dhanush Portfolio

## Features
- Responsive design (mobile-first)
- Dark/Light theme toggle with localStorage
- Hero with particle animation
- Skills tree visualization
- 5 real projects with GitHub links
- 5 certificates with modal viewer
- Resume PDF (View/Download)
- Simple social icons (Email/LinkedIn/GitHub/LeetCode)

## Quick Start
1. Open `index.html` in browser
2. Toggle theme (top-right)
3. Scroll to see animations

## Deploy
1. `git init; git add .; git commit -m "initial"; git push`
2. GitHub Pages/Netlify/Vercel

## Tech Stack
HTML5 | CSS3 (Glassmorphism) | Vanilla JS | Font Awesome
